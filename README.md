# Air Quality Index App
Tells the concentration of Air pollutants at a particular place. Also gives information about individual air pollutants.

# App components
- Has a splash screen.
- A 3 page splash screen which only appears when the app is launched for the first time.
- used breezometer API to fetch data : https://breezometer.com/products/air-quality-api/
- The data is stored using the room database.
- live data and view model is used to fetch and display data
